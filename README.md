# UConvert
Android Unit Converter


All input units will first be converted to SI then to the desired output unit.
This will introduce small conversion errors but generally those are not an issue.


## Categories
Categories of units for conversion.

- Length
- Area
- Volume (wet)
- Volume (dry)
- Weight (mass)
- Force
- Moment (torque)*
- Uniform Load
- Pressure
- Temperature
- Speed
- Acceleration
- Stiffness
- Flow Rate
- Power
- Energy*
- Density

\* Although energy and torque are ultimately the same unit we treat them differently.
